# Market Research Agent

An AI-powered agent for conducting comprehensive market research on any company. Built for PM interview prep and competitive analysis.

## What It Does

1. **Collects context** - Asks for company name, industry, stage, and your research goal
2. **Conducts live research** - Searches the web for current information
3. **Generates structured output** - Delivers a formatted research report with insights tailored to your needs

## How to Use

1. Open this folder in Claude Code
2. Start a conversation - the agent will prompt you for information
3. Answer the intake questions
4. Receive your research report

## Use Cases

- **Interview Prep**: Research a company before a PM interview
- **Competitive Analysis**: Understand a competitor's positioning
- **Market Exploration**: Learn about a new industry or space

## Default Research Scope

- Company overview and key metrics
- Top 5 competitors with positioning
- Market size (TAM/SAM/SOM)
- Recent news and developments
- Key industry trends
